개인정보 처리방식

이 앱은 완전히 로컬에서만 작동하며, 개인정보를 외부 서버로 전송하지 않습니다. 모든 할 일(Todo) 데이터는 사용자의 기기에만 저장되며, 인터넷 연결 없이도 사용할 수 있습니다. 앱에서 생성한 모든 데이터는 사용자의 기기를 벗어나지 않습니다.
Info

데이터 저장

앱의 모든 데이터는 사용자의 기기 내 로컬 스토리지(브라우저 로컬 스토리지)에만 저장됩니다. 데이터는 암호화되지 않은 상태로 저장되며, 기기의 보안은 사용자의 책임입니다. 앱을 삭제하거나 브라우저 데이터를 삭제하면 모든 데이터가 완전히 제거됩니다.
Info

제3자 정보 공유

이 앱은 어떠한 개인정보도 제3자와 공유하지 않습니다. 네트워크 통신이 발생하지 않으므로 외부 서비스나 분석 도구로 데이터가 전송되지 않습니다. 사용자의 할 일 목록, 설정 등 모든 정보는 완전히 비공개로 유지됩니다.
Info

데이터 백업 및 복구

앱에서 생성한 데이터는 사용자가 직접 백업해야 합니다. 기기 손실, 브라우저 데이터 삭제, 앱 제거 시 모든 데이터가 영구적으로 삭제됩니다. 개발자는 데이터 복구나 백업 서비스를 제공하지 않습니다.
